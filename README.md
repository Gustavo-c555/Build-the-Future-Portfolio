## Sobre o Projeto

Este é o portfólio profissional de **Gustavo Coelho**, desenvolvido com HTML, CSS e JavaScript puros. O site apresenta uma experiência imersiva com animações de scroll, efeitos de spotlight interativos, divisores SVG animados e design responsivo.

O tema "Build" reflete a filosofia de construção digital — transformar ideias em realidade através de código limpo, design elegante e atenção aos detalhes.

---

## Tecnologias

- HTML5 semântico
- CSS3 (Tailwind CSS via CDN)
- JavaScript vanilla
- SVG animados
- Google Fonts (Outfit)
- Lucide Icons (ticker de clientes)

---

## Estrutura

- **Hero Section** — Efeito de revelação com clip-path circular ao scroll
- **Clients** — Ticker infinito de parceiros
- **Services** — Cards com efeito Glass Glow e ícones SVG customizados
- **Work Gallery** — Galeria expansível com hover accordion
- **About** — Seção sobre o autor
- **Footer** — CTA e links sociais
